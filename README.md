# 💰 Simple Interest Calculator

## 📌 Description

This project is a simple interest calculator that calculates the interest based on principal amount, rate of interest, and time.

---

## 🧮 Formula Used

**Simple Interest = (P × R × T) / 100**

Where:

* P = Principal amount
* R = Rate of interest
* T = Time (in years)

---

## ⚙️ How It Works

* User enters principal, rate, and time
* The program calculates simple interest using the formula
* Displays the result
